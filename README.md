## rust_box2d

Box2D for Rust.

## Usage

You won't find a lot of information about Box2D here, look at [the official website](http://box2d.org/)
instead.

## Installation

Simply add a dependency in your Cargo.toml:

~~~~toml
[dependencies.box2d]
git = "https://github.com/Bastacyclop/rust_box2d.git"
~~~~
